1. First, the core problem Kubernetes has : 

Kubernetes runs Pods on Nodes, but:
    Nodes have limited CPU & memory
    Many Pods want to run at the same time
    Sometimes new Pods arrive when the cluster is already full

So Kubernetes must answer two questions:
    Which Pod should run first?
    If there is no space, which Pod should be removed?
