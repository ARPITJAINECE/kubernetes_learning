2. Why Kubernetes needs TLS everywhere : 
Kubernetes is distributed. Many components talk to each other.

Without TLS:
    Anyone could impersonate API server
    Anyone could read secrets
    Anyone could modify cluster state

So Kubernetes forces TLS almost everywhere.
